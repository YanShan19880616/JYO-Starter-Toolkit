# 图片目录说明

本目录用于存放新手五日速成攻略的图片。

## 所需图片文件：

- `day1.jpg` - 第一天：初入江湖 攻略图
- `day2.jpg` - 第二天：拜师学艺 攻略图
- `day3.jpg` - 第三天：肝脑涂地 攻略图
- `day4.jpg` - 第四天：脱胎换骨 攻略图
- `day5.jpg` - 第五天：名扬天下 攻略图

## 注意事项：

- 图片格式建议使用 `.jpg` 或 `.png`
- 建议图片尺寸：600x800px 或相近比例
- 如果没有放置图片，系统会自动显示占位提示图

## 如何添加图片：

1. 将你的图片文件复制到此目录
2. 确保文件名与上方列表一致（如 day1.jpg）
3. 刷新应用即可看到图片
